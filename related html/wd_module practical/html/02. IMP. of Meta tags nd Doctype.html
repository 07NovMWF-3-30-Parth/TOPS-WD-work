Importance of Meta tags and Doctype




Meta tags = > It is used for meta information of webpage for browser, so it shows properly in browsers in different devices. It is always written in head section of html.

it include following attribute.


< meta charset= UTF-8 > => it shows that html document is written in UTF-8 type character. and it is recommended by W3C.

< meta http-equiv= X-UA-Compatible content= IE=edge > = > It provide header information related to content which is IE=edge i.e. Internet Explorer Edge

< meta name= viewport content= width=device-width, initial-scale=1.0> => In name attribute viewport shows that how webpage looks in different devices and screen-size. and width shows that it uses device width with initial-scale=1.0 means 100% device width

DOCTYPE => it is declaration that shows document is HTML:5 document
