  20. What are logical and physical tags in HTML?

    21. Logical tags in html
    It is that kind of tag which tells browser what kind of text is written inside the tag. They are different from physical tag.
    example : < address > => for address or < strong > => for showing important keyword or text, used for SEO purpose.
    other tags are : < abbr >, < acronym >, < address >, < cite >, < blockquote >, < del >, < ins >, < pre >, < q >, < strong >.

    Physical tags in html
    physical tags are those tags which provide actual physical formatting to the text in html. It gives appearance to the text but not providing any information about the text.
    example : < i > => used for italic style or < b > => for bold style.
    other tags are : < b >, < big >, < i >, < small >, < sub >, < sup >, < u >
